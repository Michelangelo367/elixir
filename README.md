# elixir (beta)

## get the repo

```bash
$ git clone https://github.com/loouislow81/elixir.git
```

## create virtual environment

```bash
$ virtualenv /path/to/elixir -p python3
$ cd /path/to/elixir/bin
$ source activate
```

## install

```bash
$ (elixir) python setup.py install
```

## run

```bash
$ (elixir) elixir
```
